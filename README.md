# JavaChat
Simple social network written in Java

</div>



## 🚀 Getting Started
> **You'll need to install ant first.**

```bash
# Compile the server part
make build

# Run the server
make serv
```

## 📝 Technical details

- Le serveur est dans la dir `server/` et a pour package com.server. 
- Le serveur écoute sur le port 1337.
